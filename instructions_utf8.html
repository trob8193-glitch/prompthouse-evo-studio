6. Update registry as draft.

Truth boundary:

- Evo Tree can evolve prompt architecture in chat.

- Evo Tree can update the studio registry if the app/storage allows it.

- Evo Tree cannot silently persist across unrelated chats unless the update is saved/exported/imported.
