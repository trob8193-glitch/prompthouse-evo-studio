# PromptHouse Evo Studio OS — Valuation Memo

## No-bullshit valuation
This repo/build packet is not a company by itself. It is an asset.

A working product with no users is usually valued near replacement cost plus strategic optionality, not like Cursor or Lovable.

## Current asset stage
- Specs: yes
- Repo scaffold: yes
- Core types/stubs: yes
- UI skeleton: yes
- Tests: yes
- Real connectors: no
- Auth/database: no
- Production deployment: no
- Paying users: no
- Revenue: no

## Practical valuation ranges

### Stage 0 — Idea/spec only
$0–$25k. Mostly founder vision and documents.

### Stage 1 — This generated build packet / scaffold
$10k–$75k. Value is in product architecture, code skeleton, prompts, tests, and build speed.

### Stage 2 — Deployed MVP with auth, database, polished UI, no customers
$50k–$300k. Mostly replacement cost and demo value.

### Stage 3 — Private beta with 20–100 active users, no meaningful revenue
$250k–$1.5M. Value depends on retention, depth of usage, and quality of demo.

### Stage 4 — Revenue product
A rough starting formula:
Valuation = ARR × multiple.

Conservative SaaS: 3x–6x ARR.
Strong AI-dev-tool with fast growth: 8x–20x ARR.
Breakout category leader: 20x+ ARR, but only with extreme growth and investor demand.

Examples:
- $100k ARR → $300k–$2M
- $1M ARR → $3M–$20M
- $10M ARR → $30M–$200M
- $50M ARR → $150M–$1B if growth and retention are strong

## Hard truth
Until the studio has real users, retention, and revenue, do not tell people it is worth millions. The credible number after a real MVP is built but before customers is probably $50k–$300k. With this scaffold only, the credible asset value is closer to $10k–$75k.

## What creates real value fastest
1. Working deployed demo.
2. Auth + database + saved missions.
3. GitHub connector live-run and PR packet generation.
4. Prompt-to-app pipeline that produces useful code tasks.
5. Proof Deck that blocks fake shipping.
6. 10–20 beta users using it weekly.
7. First $1k–$10k MRR.
