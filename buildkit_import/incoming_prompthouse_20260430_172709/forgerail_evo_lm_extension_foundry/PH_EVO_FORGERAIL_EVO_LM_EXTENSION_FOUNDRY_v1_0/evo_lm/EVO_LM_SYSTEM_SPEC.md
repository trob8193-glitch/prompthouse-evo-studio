# Evo LM System Spec

## Definition
Evo LM is PromptHouse's model orchestration, studio-control, prompt-architecture, and provider-routing layer.

It is not a cloned external model. It routes to external models, coding agents, image/video generators, and local tools through PromptLink when authorized.

## Does
- chat orchestration
- prompt architecture
- code task generation
- provider routing
- image/video prompt routing
- studio automation
- proof deck enforcement
- extension generation
- cost/budget policy
- prompt fire orders
- self-repair

## Does not
- clone proprietary models
- bypass provider terms
- access external tools without credentials
- make fake deployment claims
- expose secrets

## Public responders
Only the 11 PromptHouse dev bots respond publicly by default.
