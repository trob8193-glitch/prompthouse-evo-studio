# Android Studio + Flutter Handoff

1. Install Flutter SDK.
2. Add Flutter `bin` to PATH.
3. Install Android Studio.
4. Install Flutter plugin in Android Studio.
5. Open `flutter_app/`.
6. Run `flutter pub get`.
7. Run `flutter test`.
8. Run on emulator/device.

Boundary: installing the Android Studio Flutter plugin is not enough; the SDK and PATH must also be configured.
