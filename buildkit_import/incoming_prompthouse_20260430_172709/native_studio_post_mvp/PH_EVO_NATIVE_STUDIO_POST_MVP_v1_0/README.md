# PromptHouse Evo Native Studio — Post-MVP Deployable Reference v1.0

Built: 2026-04-30

## What this is
A mobile-first app dev studio built on prompt-engineering architecture.

It includes:
- Flutter app shell
- full internal Prompt Auth policy engine
- PromptBase seed database
- saved missions
- Proof Deck
- Blueprint Engine
- Prompt Fire Orders
- simulated digital bot customers
- AI Chat Native Kernel
- Antigravity build prompt
- VS Code extension scaffold
- Android Studio / Flutter handoff
- OpenAPI + MCP connector designs

## Truth boundary
This is a real code/spec package that can be opened, run, expanded, and deployed.
It is not falsely claimed to already be in production, app stores, or your cloud.
Production deployment requires real build logs, signing, credentials, hosting, store submission, and proof receipts.

```bash
cd flutter_app

```
