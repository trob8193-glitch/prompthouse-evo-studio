# PH Evo Memory Box

This folder defines the local-first memory node. In this starter pack, memory is handled through the local gateway SQLite database. A full desktop app can be built later with Tauri or Electron without changing the memory contract.

## Modes

- off
- local_only
- private_sync
- workspace_sync
- global_contribution
- model_training_contribution
- enterprise_private

## Default

`local_only`, training off, sync off, global contribution off.
