# PH Evo Studio Builder Module v1.0

Built: 2026-04-30

## Mission
Create a PromptHouse-native visual builder where you can make custom Evo LMs, agents, references, PromptLink bridges, tests, and exports using simple blocks.

## Kid-simple version
This is like LEGO for AI agents.

You pick blocks:
1. Brain — what the agent is
2. References — what it knows
3. Tools — what it can use
4. Rules — what it cannot do
5. Approval — when it must ask you first
6. Tests — how it proves it works
7. Publish — where it goes

Then PromptHouse turns it into a recipe.

## Truth boundary
A custom Evo LM is not a new foundation model. It is a PromptHouse orchestration recipe that can route to external providers and tools only through real credentials, permissions, budgets, and proof.
