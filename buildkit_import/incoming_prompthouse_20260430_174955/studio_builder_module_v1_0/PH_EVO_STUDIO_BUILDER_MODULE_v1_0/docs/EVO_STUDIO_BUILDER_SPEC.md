# Evo Studio Builder Spec

## Purpose
Evo Studio Builder lets users create custom Evo LMs, agents, teams, workflows, PromptLink bridges, ForgeRail rails, and Extension Foundry modules through a simple visual builder.

## Core screens
1. Type Picker
2. Block Library
3. Flow Canvas
4. Recipe Inspector
5. References Panel
6. Tool Permissions Panel
7. Tests Panel
8. Proof Panel
9. Export Panel

## Block types
- Start
- Evo LM
- Agent
- Team
- References
- PromptBase
- PromptLink
- Tool
- Guardrail
- User Approval
- If / Else
- Loop
- Test
- Proof Receipt
- Export / Publish
- End

## Outputs
- Evo Agent Recipe YAML
- AI chat kernel
- Antigravity prompt
- VS Code task pack
- Flutter screen/service scaffold
- PromptLink adapter manifest
- ForgeRail rail
- ExtensionCapsule
- Proof checklist

## No-fake rules
- No external provider access without real credentials.
- No repo write without approval.
- No deployment claim without proof.
- No app-store claim without receipt.
- No secret values in prompts or logs.
