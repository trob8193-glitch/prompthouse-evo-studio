# Evo Studio Builder Block Library

## Start
Begins the workflow.

## Evo LM
Creates a custom Evo LM orchestration recipe.

## Agent
Creates a single-purpose agent.

## Team
Creates a group of agents with roles.

## References
Connects files, PromptBase records, URLs, images, code folders, and prior missions.

## PromptBase
Pulls saved prompt templates.

## PromptLink
Connects to external providers/tools through permissioned adapters.

## Tool
Adds a usable capability like GitHub, Flutter, file search, image/video generation, database, browser, MCP, or OpenAPI.

## Guardrail
Adds a rule that blocks unsafe or unsupported behavior.

## User Approval
Adds “ask me first” gates.

## If / Else
Adds simple logic.

## Loop
Repeats a check until it passes or gets blocked.

## Test
Creates tests for behavior.

## Proof Receipt
Requires evidence before status becomes verified.

## Export / Publish
Exports the recipe to PromptHouse, AI chat, Antigravity, VS Code, Flutter, or another target.

## End
Finishes the workflow.
