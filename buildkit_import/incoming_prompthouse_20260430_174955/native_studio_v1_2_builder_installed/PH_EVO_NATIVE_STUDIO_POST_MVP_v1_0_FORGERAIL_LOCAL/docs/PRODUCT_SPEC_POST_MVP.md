# Product Spec — PromptHouse Evo Native Studio

PromptHouse is a studio runtime:
- Prompt Auth controls prompt actions.
- PromptBase stores reusable prompt architecture.
- Fire Orders turn intent into execution commands.
- Blueprints turn commands into app structures.
- Proof Deck prevents fake completion.
- Bot Customers simulate market and user feedback.
- Compatibility Hub exports to AI chats, Antigravity, Flutter, VS Code, and Android Studio.

Post-MVP features:
saved missions, PromptBase, prompt auth console, proof deck, blueprint generator, fire orders, bot customer simulation, mobile app shell, connector contracts, cross-IDE compatibility, AI-chat portable kernel.
